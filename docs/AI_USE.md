# AI Use

AI-assisted coding was used to help design, implement, test, and document this
prototype. All generated code and documentation were reviewed and verified
through automated tests and direct workflow testing.

At runtime, the deployed application uses local PaddleOCR models for text
detection and recognition. Deterministic code performs field comparisons and
the selected TTB-backed health-warning check. An AI model does not approve or
reject an application; the compliance officer makes the final decision.
